# 🚀 SentryMirror Roadmap

This document outlines the planned future improvements and feature requests for SentryMirror.

## 🛠️ Phase 1: Core Enhancements
- [ ] **WebSocket Support**: Capture and simulate WebSocket traffic for real-time applications.
- [ ] **Advanced Schema Inference**: Use AI/ML models to better guess database relations and data types from API responses.
- [ ] **Authentication Support**: Built-in handling for OAuth2, JWT, and session cookies to crawl protected areas more easily.
- [ ] **Custom Proxy Support**: Allow routing traffic through external proxies for better anonymity and bypassing restrictions.

## 🌐 Phase 2: Protocol & Framework Expansion
- [ ] **gRPC Support**: Capture and simulate gRPC traffic.
- [ ] **GraphQL Deep Dive**: Better introspection and simulation of GraphQL endpoints, including mutations.
- [ ] **Single Page Application (SPA) Optimizations**: Improved handling for complex React/Vue/Angular state transitions.

## 🛡️ Phase 3: Security & Performance
- [ ] **Automated Vulnerability Scanning**: Integrate basic SQL injection and XSS probes based on inferred schemas.
- [ ] **Distributed Crawling**: Support for distributed crawling to speed up analysis of large sites.
- [ ] **Detailed Performance Analytics**: Track and report on API response times and resource loading performance.

## 📊 Phase 4: Integration & UX
- [ ] **Web UI Dashboard**: A local web interface to visualize crawl results, database schemas, and manage the simulated API.
- [ ] **CI/CD Integration**: Plugins for popular CI/CD tools to run security audits and API regression tests automatically.
- [ ] **Export to Postman/Insomnia**: Generate collections for popular API testing tools.
